# Security Policy

## Intended Use

Dark Night Rises CTF is an **educational platform only**. All vulnerabilities demonstrated are simulated entirely within the browser using JavaScript. No real servers, databases, or external systems are involved.

## Reporting a Vulnerability in this Repo

If you find an actual security vulnerability in this repository's code (e.g. a way to use it to harm real systems), please open a GitHub Issue marked `[SECURITY]`.

## Disclaimer

The techniques demonstrated in this CTF platform (SQL injection, XSS, SSRF, etc.) are shown for **learning purposes only**. Applying these techniques against systems you do not own or have explicit written permission to test is illegal in most jurisdictions.

The authors accept no responsibility for misuse of the knowledge gained from this platform.

## Ethical Use

- ✅ Use on your own systems
- ✅ Use in authorised penetration tests
- ✅ Use in educational/training environments
- ❌ Do NOT use against systems without permission
- ❌ Do NOT use to harm, disrupt, or steal from others
